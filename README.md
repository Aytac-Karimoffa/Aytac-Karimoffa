## ✨ О проекте

Это приложение позволяет:
- Регистрироваться и входить в аккаунт
- Создавать личные заметки
- Просматривать и удалять свои заметки
- Искать заметки

Проект сделан с разделением на **Frontend** и **Backend** (Java Spring Boot).

---

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Java + Spring Boot
- **База данных**: H2 (встроенная)
- **Инструменты**: Git, GitHub, IntelliJ IDEA
